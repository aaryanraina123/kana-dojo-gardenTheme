## ğŸ“– Tentang KanaDojo

KanaDojo adalah platform pembelajaran bahasa Jepang berbasis web yang menarik yang membuat penguasaan Hiragana, Katakana, Kanji, dan Kosakata menjadi menyenangkan dan intuitif. Dibangun dengan fokus pada estetika, kustomisasi, dan pembelajaran efektif, KanaDojo menyediakan lingkungan pelatihan yang imersif untuk pembelajar bahasa Jepang di semua tingkat.

Baik Anda baru memulai dengan suku kata kana dasar atau mempersiapkan ujian JLPT dengan kanji dan kosakata tingkat lanjut, KanaDojo menawarkan pengalaman belajar yang efisien dan bebas gangguan yang beradaptasi dengan preferensi dan gaya belajar Anda.

### âœ¨ Fitur Utama

#### ğŸ¯ **Tiga Dojo Pelatihan**

- **Dojo Kana** - Kuasai suku kata Hiragana dan Katakana dengan grup dasar, dakuon, yoon, dan bunyi asing
- **Dojo Kanji** - Pelajari karakter kanji penting yang diorganisir berdasarkan tingkat JLPT (N5, N4, N3, N2)
- **Dojo Kosakata** - Bangun kosakata bahasa Jepang Anda dengan koleksi kata yang dikurasi berdasarkan tingkat kemahiran

#### ğŸ® **Empat Mode Permainan Dinamis**

Setiap dojo mendukung empat mode pelatihan yang menarik untuk memperkuat pembelajaran:

1. **Pick** - Pilihan ganda: Pilih romanisasi/terjemahan yang benar untuk karakter yang ditampilkan
2. **Reverse-Pick** - Pilihan ganda terbalik: Pilih karakter yang benar untuk romanisasi/terjemahan yang diberikan
3. **Input** - Input teks: Ketik romanisasi/terjemahan yang benar
4. **Reverse-Input** - Input teks terbalik: Ketik karakter yang benar

#### ğŸ¨ **Kustomisasi Ekstensif**

- **100+ Tema** - Pilih dari koleksi tema terang dan gelap yang indah, atau gunakan fitur tema acak
- **28 Font Jepang** - Pilih dari berbagai tipografi Jepang otentik untuk menyesuaikan preferensi estetika Anda
- **Efek Suara** - Nikmati suara umpan balik UI yang memuaskan yang dapat diaktifkan/nonaktifkan
- **Opsi Tampilan** - Beralih antara tampilan Romaji/Bahasa Inggris dan Kana/Kanji di menu pemilihan
- **Tombol Pintas** - Pintasan keyboard untuk pelatihan yang efisien (dapat dinonaktifkan)

#### ğŸ“Š **Pelacakan Kemajuan**

- Umpan balik waktu nyata dengan penghitung benar/salah
- Pelacakan streak untuk mempertahankan motivasi
- Statistik untuk memantau kemajuan belajar Anda

#### ğŸŒ **Pengalaman Web Modern**

- Desain responsif penuh yang berfungsi di desktop, tablet, dan mobile
- Tidak perlu instalasi - berlatih di mana saja dengan koneksi internet
- Antarmuka bersih dan minimalis yang membuat Anda fokus pada pembelajaran
- Animasi dan transisi halus yang didukung oleh Framer Motion

<a id="arsitektur"></a>

## ğŸ—ï¸ Arsitektur

KanaDojo mengikuti **arsitektur berbasis fitur** yang mengorganisir kode berdasarkan fungsionalitas daripada berdasarkan jenis file. Pendekatan modular ini meningkatkan maintainability, skalabilitas, dan pengalaman pengembang.

### Struktur Utama

- **`features/`** - Modul mandiri berdasarkan fungsionalitas (kana, kanji, vocabulary, statistics, achievements, themes, academy, cloze)
- **`shared/`** - Komponen, hooks, utilitas, dan tipe yang dapat digunakan kembali di seluruh fitur
- **`core/`** - Infrastruktur fundamental (i18n, analytics)
- **`app/`** - Next.js App Router dengan halaman dan layout

Setiap fitur berisi komponen, store, data, tipe, dan logika bisnis sendiri, memungkinkan pengembangan independen dan pemahaman kode yang lebih mudah. Untuk informasi arsitektur terperinci, lihat [docs/ARCHITECTURE.md](./ARCHITECTURE.md).

### Instalasi

1. **Clone repositori**

   ```bash
   git clone https://github.com/lingdojo/kanadojo.git
   cd kanadojo
   ```

2. **Instal dependensi**

   ```bash
   npm install
   ```

3. **Jalankan server pengembangan**

   ```bash
   npm run dev
   ```

4. **Buka browser Anda**
   Navigasi ke [http://localhost:3000](http://localhost:3000)

```bash

npm start
```

```bash
# Generate sitemap (berjalan otomatis setelah build)
```

### Konsep Kunci

---

#### Arsitektur Berbasis Fitur

KanaDojo menggunakan pola arsitektur modular di mana setiap fungsionalitas bersifat independen:

- **Enkapsulasi**: Setiap fitur berisi semua yang dibutuhkan (komponen, state, data, logika)
- **Barrel Exports**: Setiap modul mengekspor API publik melalui `index.ts`
- **Type Safety**: TypeScript dengan alias path (`@/features/*`, `@/shared/*`, `@/core/*`)
- **Separation of Concerns**: Fitur yang terisolasi, kode bersama yang dapat digunakan kembali, inti fundamental

#### Alur Manajemen State

1. Pengguna memilih konten di komponen menu
2. Pilihan disimpan di store Zustand (store khusus fitur)
3. Komponen pelatihan membaca dari store untuk menghasilkan pertanyaan
4. Statistik dilacak dan dipersisten di `features/statistics/store`
5. Preferensi pengguna disimpan di `features/themes/store` dengan persistensi localStorage

#### Arsitektur Komponen

- **Komponen Fitur**: Spesifik untuk setiap fungsionalitas (kana, kanji, vocabulary)
- **Komponen Bersama**: UI yang dapat digunakan kembali di seluruh fitur (Game, Modals, AudioButton)
- **Komponen Layout**: Navigasi, menu utama, struktur halaman

#### Organisasi Data

- **Kana**: Diorganisir berdasarkan tipe (hiragana/katakana) dan grup (base, dakuon, yoon, foreign)
- **Kanji**: Diorganisir berdasarkan tingkat JLPT (N5-N2), dengan bacaan dan makna
- **Vocabulary**: Diorganisir berdasarkan tingkat JLPT dan jenis kata (kata benda, kata kerja, dll.)

#### Implementasi Mode Permainan

Setiap mode permainan adalah rute dinamis (`/[contentType]/train/[gameMode]`) yang:

1. Membaca konten yang dipilih dari store yang sesuai
2. Menghasilkan pertanyaan acak dari pilihan
3. Memberikan umpan balik langsung
4. Melacak statistik (benar, salah, streak)
